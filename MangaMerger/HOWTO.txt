==============================
 MANUAL MANGA COMBINER - SETUP
==============================

This guide explains how to run the Manga Combiner web app
on a normal PC or on Unraid using the "Python3 for Unraid" plugin.

------------------------------
 1. INSTALL DEPENDENCIES
------------------------------
Open a terminal or command prompt inside the extracted folder.

Run these commands:

pip install flask
pip install PyPDF2
pip install reportlab
pip install pillow

(These only need to be done once.)


------------------------------
 2. CONFIGURE PATHS
------------------------------
Open the file:  config.json

Example:
{
  "main_path": "C:/Manga",
  "export_path": "C:/Manga_Exports",
  "port": 3636
}

Change:
â€¢ main_path  = where your manga folders are stored
â€¢ export_path = where finished combined files should go
â€¢ port = website port (default 3636)


------------------------------
 3. RUN ON WINDOWS / PC
------------------------------
1. Run:
   python app.py
2. Wait until it says: "Running on http://127.0.0.1:3636"
3. Open a web browser and go to:
   http://127.0.0.1:3636

To stop it, press CTRL + C in the terminal.


------------------------------
 4. RUN ON UNRAID
------------------------------
1. Install the plugin:
   "Python3 for Unraid by desertwitch"

2. Copy the entire MangaCombiner folder to your unraid server. Anywhere works

3. Add this to the autoexec.sh
```
# 1. Go to the project folder (Change to your location)
cd /mnt/user/##/##/MangaMerger || exit

# 2. Reinstall dependencies if needed (persists across reboots)
pip install --upgrade flask PyPDF2 reportlab pillow >/dev/null 2>&1

# 3. Start the web server in the background
#    Port is set in config.json (default 3636)
python3 app.py >/dev/null 2>&1 &
````

(There is a log file that will be made)

4. In your browser, go to your Unraid server IP and port 3636.
   Example:
   http://192.168.1.10:3636

To stop it, run kill_server.py
